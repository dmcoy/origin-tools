# Origin Tools
A blender add-on for quickly manipulating object origins
<img width="300" alt="image" src="/readme_images/origin-tools-panel.png" />

## Installation
Download the .zip file from release and install from disk in Blender under **Preferences** --> **Add-ons**

## Usage
The Origin Tools add-on is in the 3D viewport. Press `n` to show the sidebar panels. This add-on can be used in object mode and edit mode. At least one object must be selected to use the operations. "Set origin to selection" requires being in edit mode.

## Features

**Change Origin Orientation**

This changes the orientation of the origin while preserving the object's position and rotation. This can be useful swapping origin axes around (e.g. swapping the Z for Y axis).

**Set Origin**

These buttons are shortcuts to the set origin options. These will work for all selected objects.
